# StockMind v2 - 产品需求规格书

> 项目代号：**StockMind AI跟投版**
> 定位：小白用户的AI炒股助手——AI告诉你买什么，为什么买，你一键跟
> 核心理念：「上善若水」——顺势而为，AI替你读懂市场

---

## 一、目标用户

- **核心用户**：想炒股但不懂技术分析的小白上班族
- **用户痛点**：看不懂K线、不知道买哪只、担心亏损
- **解决方案**：AI每日推荐 + 一键跟投 + 全程解释

---

## 二、核心产品逻辑

```
用户打开App → 看到「今日AI推荐」→ 读懂理由 → 一键跟投 → AI持续跟踪 → 推送止盈/止损
```

**关键设计原则：**
1. **第一屏就要有答案** —— 不需要用户自己探索，AI直接给结论
2. **每条推荐都要有解释** —— 老人也能看懂"为什么买这只"
3. **操作极简** —— 跟投只需1次点击
4. **全程透明** —— 买入/止损/止盈全部推送，不让用户蒙在鼓里

## 四、AI推荐逻辑（核心）

### 每日选股流程（简化为3步）
```
Step 1：价值过滤（格雷厄姆）
  PE: 5~50
  ROE ≥ 8%
  负债率 ≤ 60%
  → 候选池约200只

Step 2：趋势确认（利弗莫尔）
  MA5 > MA20（多头）
  成交量放量 ≥ 1.3倍
  MACD > 0
  → 候选池约20只

Step 3：资金验证（跟庄）
  北向资金近5日净买入
  主力资金净流入
  → TOP 5 输出
```

### AI理由生成
每条推荐必须附带：
1. **资金面**（北向/主力/杠杆资金）
2. **技术面**（哪个指标触发）
3. **基本面**（为什么值得）
4. **风险提示**（哪里需要注意）
5. **操作建议**（仓位/止损/目标价）

| 数据 | 来源 |
|------|------|
| 实时行情 | akshare（东方财富/新浪） |
| K线数据 | akshare（日线/60分钟） |
| 北向资金 | akshare（陆股通） |
| 主力资金 | akshare（板块资金流） |
| 基本面 | akshare（财务数据缓存） |
| 指数数据 | akshare（沪深300/上证） |

**缓存策略**：行情数据缓存5分钟，基本面数据缓存24小时

### 字体
- 数字：等宽（tabular-nums），大字号突出

- 页面入场：fadeInUp，300ms，stagger 100ms
- 卡片悬停：边框发光 + scale(1.01)
- 数据更新：数字跳动动画
- 盈亏变化：绿色/红色闪烁提示

- 「历史胜率 XX%」标签
- 「XX人正在跟投」实时数字

## 九、后端 API 设计（Worker.js）

```javascript

GET  /api/health               → 健康检查

GET  /api/stocks/:code         → 股票详情（K线/指标/AI理由）
GET  /api/portfolio            → 我的持仓（含盈亏）
POST /api/portfolio/buy        → 跟投买入
POST /api/portfolio/sell       → 卖出
POST /api/chat                 → AI问答（MiniMax）
GET  /api/ai/signal            → 今日推荐信号
```
