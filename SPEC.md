# 炒股量化系统 v2.0 - 完整需求规格书

> 项目代号：**StockMind**（股票智慧体）
> 目标：让普通散户也能用机构级策略选股、跟单、复盘
> 核心理念：道家"上善若水"——顺势而为，不逆市而动

---

## 一、核心目标

| 指标 | 目标值 | 说明 |
|------|--------|------|
| 年化收益率 | ≥10% | 跑赢通胀 + 稳定增值 |
| 最大单笔亏损 | ≤8% | 硬止损线 |
| 资金规模 | 10万以内 | 轻量资金试水 |
| 操作模式 | 半自动（AI信号 + 手动确认） | 人机协同，AI给建议，人做决策 |
| 交易周期 | 周/月经趋势为主 | 不频繁交易，降低摩擦成本 |
| 数据验证 | 先回测，再模拟盘，最后实盘 | 三步验证法 |

### 3.1 K线分析模块（P0 - 核心）

**目标**：交互式K线图表，支持多指标叠加

| 坐标模式 | 普通坐标 / 对数坐标 |
| 缩放平移 | 鼠标滚轮缩放，拖拽平移 |
| 数据范围 | 最近1年 / 3年 / 5年 / 自定义 |

#### 技术指标（全部支持）
| 指标 | 参数 | 显示方式 |
|------|------|----------|
| MA均线 | 5/10/20/60/120/250 | 线条叠加 |
| MACD | 12/26/9 | 副图（柱状+信号线）|
| RSI | 6/12/24 | 副图（0-100区间）|

### 3.2 智能选股模块（P0 - 核心）

**目标**：从全市场5000+股票中，每日筛选出TOP 10候选股

#### 筛选流程（6步流水线）

```
Step 1：数据获取（akshare）
        ↓ 全市场OHLCV + 财务数据
Step 2：价值筛选（基础过滤）
        ├ PE: 5 ≤ PE ≤ 50（放宽原PE<15）
        ├ PB ≤ 5
        ├ ROE ≥ 8%（下调原10%）
        ├ 净利润增速 ≥ 5%
        ├ 营收增速 ≥ 5%
        ├ 负债率 ≤ 60%
        ├ 商誉/净资产 ≤ 30%
        ├ 上市时间 ≥ 1年
        └ 排除：ST / 次新股 / 涨跌停（封板买不进）
        ↓ 候选池A（约200-500只）
Step 3：趋势确认（技术面过滤）
        ├ MA5 > MA20（多头排列）
        ├ 20日涨幅 > 0
        ├ 成交量 > 20日均量 × 1.3
        ├ MACD > 0 或 金叉（当日）
        └ RSI < 75 且 > 30
        ↓ 候选池B（约20-50只）
Step 4：资金验证（加分项）
        ├ 北向资金（陆股通）近5日净买入
        ├ 主力资金近3日净流入
        └ 大宗交易折价 < 5%
        ↓ 候选池C（约10-20只）
Step 5：评分排名
        评分维度：
        ├ 动量得分（20日涨幅排名）× 20%
        ├ 趋势得分（均线多头天数）× 20%
        ├ 资金得分（北向+主力净流入排名）× 20%
        ├ 基本面得分（ROE+增速综合排名）× 20%
        └ 形态得分（突破有效性）× 20%
        ↓ TOP 10
Step 6：风控检查
        ├ 行业分散（同一行业≤2只）
        ├ 仓位上限（单只≤20%）
        └ 总持仓≤5只
        ↓ 今日买入信号输出
```

### 3.3 策略回测模块（P0 - 核心）

**目标**：验证策略有效性，支持单策略/多策略对比

#### 回测引擎
- 框架：Backtesting.py（轻量、活跃维护、Pandas原生）
- 回测周期：任意自定义（默认近2年）
- 初始资金：自定义（默认10万）
- 手续费：万3买卖 + 万0.5印花税 + 0.5%滑点

#### 回测功能
| 功能 | 说明 |
|------|------|
| 单策略回测 | 选定一只/多只股票，应用完整策略 |
| 多策略对比 | 最多4个策略同时回测，对比收益曲线 |
| 参数优化 | 对关键参数（MA周期/RSI阈值）做网格搜索 |
| 样本外验证 | 用训练集最优参数，在测试集验证 |
| Monte Carlo模拟 | 1000次随机模拟，评估策略稳定性 |

## 四、交易策略体系（融合四大投资大师）

### 4.1 策略架构：三层过滤

```
┌─────────────────────────────────────────────────────────────┐
│                    三层过滤选股体系                           │
│                                                             │
│  【第一层：价值过滤】—— 格雷厄姆 + 巴菲特                    │
│  ├ PE: 5~50（成长股可接受更高）                             │
│  ├ PB ≤ 5（优质公司可适当放宽）                             │
│  ├ ROE ≥ 8%（持续盈利能力）                                │
│  ├ 净利润增速 ≥ 5%（成长性验证）                            │
│  ├ 营收增速 ≥ 5%                                            │
│  ├ 负债率 ≤ 60%（财务安全）                                │
│  ├ 经营现金流 > 0（真实盈利）                               │
│  └ 商誉/净资产 ≤ 30%（避雷）                                │
│                                                             │
│  【第二层：趋势过滤】—— 利弗莫尔 + 彼得·林奇                 │
│  ├ MA5 > MA20（上升趋势确认）                               │
│  ├ 20日涨幅 > 0（近期动量）                                │
│  ├ 成交量 > 20日均量 × 1.3（资金认可）                     │
│  ├ MACD柱状由绿转红（趋势强化）                             │
│  ├ RSI: 30~75（不过热，可介入）                             │
│  └ 股价 > 20日最高价（突破确认）                           │
│                                                             │
│  【第三层：资金过滤】—— 跟随主力                             │
│  ├ 北向资金（陆股通）近5日净买入                            │
│  ├ 主力资金近3日净流入                                      │
│  └ 融资余额上升（杠杆资金认可）                             │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### 4.2 买入规则（优先级排序）

```
【开仓信号】必须同时满足：
  ✓ 第一层：价值过滤通过
  ✓ 第二层：趋势过滤通过（至少4/6条件）
  ✓ 第三层：资金过滤通过（至少1/3条件）

【仓位计算】（彼得·林奇GARP策略）
  单只仓位 = 总资金 × (20% / ATR百分比)
  上限：单只 ≤ 20%
  下限：单只 ≥ 5%
  最小单位：1手（100股）

【止损规则】
  买入后当即设定初始止损 = 买入价 × 0.92（-8%）
  动态调整：每上涨10%，止损线上移5%（锁住更多利润）
```

### 4.3 卖出规则（优先级排序）

```
【触发以下任一条件即卖出】：

  1. ⚠️ 硬止损（无条件）
     亏损达到 -8%，次日开盘清仓

  2. 🔴 趋势破坏（次日执行）
     收盘价 < MA20（20日均线）
     或 MA5 < MA10（均线死叉）

  3. 🟡 动态止盈（触发后跟踪）
     盈利 > 20% 后，进入止盈跟踪模式
     收盘价从近期高点回撤 > 5% → 止盈

  4. ⏰ 时间止损（到期检查）
     持有 > 30天 且 盈利 > 5% → 止盈
     持有 > 30天 且 亏损 → 止损

  5. 📉 市场切换（熊市信号）
     沪深300跌破20周均线 → 清仓所有持仓
     （彼得·林奇：牛熊转换时，保住利润优先）
```

### 4.4 牛熊判断规则

```
【市场状态判定】（每周五收盘后更新）

  ┌─────────────────────────────────────────┐
  │  🐂 牛市                              │
  │  条件：沪深300 > 20周均线 AND           │
  │        沪深300 20周涨幅 > 0             │
  │  操作：标准仓位（最多5只，总仓位≤80%）   │
  ├─────────────────────────────────────────┤
  │  🐻 熊市                              │
  │  条件：沪深300 < 20周均线 AND            │
  │        沪深300 20周跌幅 > 5%            │
  │  操作：清仓或极轻仓（≤1只，仓位≤20%）   │
  ├─────────────────────────────────────────┤
  │  ⚖️ 震荡市                            │
  │  条件：上述两种都不满足                  │
  │  操作：标准仓位减半（≤2只，仓位≤40%）   │
  └─────────────────────────────────────────┘
```

## 八、API接口设计

### 8.1 股票数据接口

```
GET  /api/stock/{code}
     获取单只股票基本信息
     返回：名称、行业、市值、PE、PB、ROE等

GET  /api/stock/{code}/kline
     获取K线数据
     参数：ktype（日/周/月）, start, end
     返回：OHLCV数组

GET  /api/stock/{code}/indicators
     获取技术指标
     参数：indicators（MA,MACD,RSI等逗号分隔）
     返回：各指标数值

GET  /api/stock/{code}/moneyflow
     获取资金流向
     返回：北向资金、主力净流入等
```

### 8.2 选股接口

```
GET  /api/picker/signals
     获取今日选股信号
     返回：TOP 10股票 + 评分 + 建议

GET  /api/picker/stockpool
     获取当前候选股票池
     参数：page, page_size
     返回：满足条件的股票列表

POST /api/picker/custom
     自定义筛选
     参数：PE范围、ROE、趋势条件等
     返回：符合条件的股票
```

### 8.3 回测接口

```
POST /api/backtest/run
     执行回测
     参数：stock_code, strategy, start, end, capital
     返回：回测结果（收益曲线、买卖点、统计指标）

GET  /api/backtest/compare
     多策略对比
     参数：strategy_ids[]
     返回：对比图表数据

GET  /api/backtest/history
     获取历史回测记录
     返回：回测历史列表
```

### 8.4 持仓接口

```
GET  /api/portfolio
     获取当前持仓
     返回：持仓列表 + 总盈亏

POST /api/portfolio/add
     添加持仓记录
     参数：code, buy_date, buy_price, quantity

DELETE /api/portfolio/{id}
     删除持仓记录

GET  /api/portfolio/risk
     获取风控状态
     返回：各持仓风险状态、止损价
```

### 8.5 AI接口

```
POST /api/ai/generate-strategy
     AI策略生成
     参数：description（自然语言描述）
     返回：策略代码 + 说明

POST /api/ai/reverse-strategy
     AI策略反推
     参数：code, start, end
     返回：策略描述 + 代码

POST /api/ai/consult
     AI股票问诊
     参数：codes[]（持仓股票列表）
     返回：诊断报告

GET  /api/ai/daily-summary
     获取每日复盘
     返回：AI生成的复盘内容
```

## 十、风险提示

1. **策略风险**：任何策略都不能保证盈利，回测≠实盘
2. **数据风险**：akshare免费数据可能有延迟或错误
3. **滑点风险**：实盘成交价与信号价可能有差异
4. **心理风险**：手动操作可能违背策略信号
5. **黑天鹅**：突发政策/利空可能导致远超8%的亏损

## 十一、爷的操作手册

```
⏰ 16:00（收盘后）
   系统自动：获取数据 → 运行选股 → 发送飞书信号
   ↓

   ↓

   ↓
   爷操作：在系统录入持仓（买入价/数量）

⏰ 16:30
   系统自动：检查持仓 → 发送风控报告
   ↓
   爷收到：持仓状态（正常/警戒/止损）
   ↓
   触发止损？→ 爷在中山证券手动卖出

⏰ 21:00
   系统自动：生成AI每日复盘
   ↓
   爷收到：复盘报告 → 阅读 → 为明天做准备
```
