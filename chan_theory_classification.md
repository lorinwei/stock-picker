## 核心公式总结

**最终操作函数**（课程68）：
```
f(X) = 买点，X = 买入
f(X) = 卖点，X = 卖出
f(X) = 持有，X = 买卖点之间
```

**三大客观支点**（课程84）：
1. 走势的不可重复性 → 不可绝对预测
2. 自同构性结构的绝对复制性 → 绝对可操作
3. 理论的纯逻辑推导 → 结论绝对有效

**递归定义框架**（课程84）：
- f1（分型→笔→线段→最低级别中枢）= a0 → a1
- f2（最低级别以上，中枢递归）= an → an+1
